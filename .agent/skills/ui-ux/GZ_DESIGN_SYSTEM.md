# GZ Design System (Luxury Editorial Hospitality)

## Sovereign Palette
- **Gold**: `#C5A059`
- **Deep Black**: `#1A1A1A`
- **Pure White**: `#FFFFFF`

## Typography
- **Primary / Header (Serif)**: `Playfair Display`
- **Body (Sans-Serif)**: `Inter`
- **Usage Strategy**: High-contrast pairing. The sophisticated Serif applies exclusively to headings to mirror the original site's luxury ethos. The clean Sans-Serif guarantees readability for dense editorial body copy.

## Editorial Layout
- **Grids**: Strictly avoid standard 1:1 grids. Implement staggered "Magazine Style" layouts.
- **Aspect Ratios**: Ensure organic flow utilizing varied image shapes to draw the eye progressively down the viewport.

## Performance UX (Cold Start Masking)
- **Skeleton Shimmers**: Leverage Gold/Gray skeleton blocks (`animate-pulse`) tailored for all Directus-loaded elements. This masks any latency generated by Cloud Run cold-starts transforming delay into a sophisticated loading experience.
